## run

[install meteor](https://www.meteor.com/install) (OSX/linux: `curl https://install.meteor.com/ | sh` )

```bash
$ npm install 
$ npm run dev
```

app will run on http://localhost:8100/

## dev
for development purpose, run as written before + watch for changes in /src:
```bash
$ gulp watch
```
